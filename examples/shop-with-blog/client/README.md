# Shop with Blog Example

## How to use
Download the example [or clone the whole project](https://github.com/deity-io/falcon.git):

Install it and run:

```bash
yarn install
yarn dev
```

Run tests:

```bash
yarn test
```

Build and run production package:

```bash
yarn build
yarn prod
```
